Travel Booking Site
=====================

A simple travel booking website built using HTML5. This project demonstrates basic web development skills, including structuring content, adding images, hyperlinks, and formatting with headings and lists.

---------------------------------------
Project Structure
---------------------------------------
- index.html   -> Main travel booking page with packages
- home.html    -> Home page
- booknow.html -> Booking confirmation and payment options
- contactus.html -> Contact information

---------------------------------------
Features
---------------------------------------
- Simple navigation between pages
- Display of available travel packages with images
- Booking page with payment options
- Contact information for support

---------------------------------------
Requirements
---------------------------------------
To run this project, you will need:
- VS Code (or any code editor)
- A modern web browser (Chrome, Edge, Firefox, etc.)
- Git & GitHub account

---------------------------------------
How to Run
---------------------------------------
1. Clone the repository:
   git clone https://github.com/your-username/travel-booking-site.git

2. Open the project folder in VS Code.

3. Open index.html (or home.html) in your browser to view the site.
